$$<M, w> \in A_{TM} \Leftrightarrow f(<M, w>)=<M_1> \in F_{TM}$$
2. TM $M_1$ works as follows when string x is inputted\\
$M_1$ simulates M on w\\
if M accepts w, then if x is "Fall", then $M_1$ accepts x, otherwise, $M_1$ rejects x\\
